Властивість font-size

    Властивість  font-size  змінює розмір шрифта.
    Розмір шрифта задається за допомогою значення та одиниць виміру (em,px,pt,%)

Синтаксис:

CSS

 p{ 
        font-size: 20px; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!

Властивість font-weight

Властивість  font-weight встановлює насиченість шрифту. 
Може приймати наступні значення:
  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме що і 700
  • lighter - змінює насиченість відносно батьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно батьківського стилю в більшу сторону
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис:

CSS

 p{ 
        font-weight: bolder; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!

Властивість font-style

Визначає зображення шрифту - звичайне, курсивне або похило.
    Коли для тексту встановлено курсивне або похилого зображення, браузер звертається до системи для пошуку відповідного шрифту.        
    Якщо шрифт не знайдено, браузер використовує спеціальний алгоритм для імітації потрібного вигляду тексту.
  • normal - Звичайне зображення тексту.
  • italic - Курсивне зображення.
  • oblique - Похиле накреслення. Курсив і похилий шрифт при всій їхній схожості не одне й те саме.
Курсив це спеціальний шрифт, що імітує рукописний, похилий утворюється шляхом нахилу звичайних знаків вправо.

Синтаксис:

CSS

 p{ 
        font-style: oblique; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!

Властивість font-family

Властивість  font-family визначає, який шрифт використати. 
Може приймати наступні значення:
  • serif - шрифти із засічками (антиквенні), типу Times;
  • sans-serif - рубані шрифти (шрифти без засічок чи гротески), типовий представник - Arial;
  • monospace - моноширинні шрифти, ширина кожного символу у такому сімействі однакова (шрифт Courier).

Синтаксис:

CSS

 p{ 
        font-family:sans-serif; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!

Властивість color

Визначає колір тексту елемента.
    Ключові слова для кольору - це ідентифікатори без урахування регістру, що представляють певний колір, наприклад red, blue, black або lightseagreen. 
    Хоча назви більш менш описують їх відповідні кольори, вони по суті штучні, без строгого обґрунтування використовуваних імен.

Синтаксис:

CSS

 p{ 
        color: red; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world

Властивість text-transform

Керує перетворенням тексту елемента у заглавні або прописні символи. 
Коли значення відрізняється від none, регістр вихідного тексту буде змінено.
  • capitalize - Перший символ кожного слова в реченні буде великим. Інші символи свого вигляду не змінюють.
  • lowercase - Усі символи тексту стають малими (нижній регістр).
  • uppercase - Усі символи тексту стають великими (верхній регістр).
  • none - Не змінює регістр символів.

Синтаксис:

CSS

 p{ 
        text-transform: uppercase; 
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!

Властивість text-decoration

    Додає оформлення тексту як його підкреслення, перекреслення чи лінії над текстом. 
    Одночасно можна застосувати більше одного стилю, перераховуючи значення через пропуск.
  • blink - Встановлює миготливий текст.
  • line-through - Створює перекреслений текст.
  • overline - Лінія проходить над текстом.
  • underline - Встановлює підкреслений текст.
  • none - Скасовує всі ефекти, у тому числі й підкреслення посилань, яке задано за замовчуванням.

Синтаксис:

CSS

 p{
        text-decoration: overline underline;
    }

HTML

<p>Hello, world!</p>

Результат:

Hello,world!
